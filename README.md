# OIBSIP_Wine_Quality_Prediction
Wine Quality Prediction using Machine Learning with Random Forest Classifier, Python, Pandas, Scikit-learn, Matplotlib, and Seaborn.

## 📌 Project Overview

This project aims to predict the quality of wine using Machine Learning techniques. The dataset contains various physicochemical properties of wine, and a Random Forest Classifier is used to classify wine quality based on these features.

---

## 🎯 Objectives

- Analyze the wine quality dataset
- Perform Exploratory Data Analysis (EDA)
- Visualize feature relationships
- Train a Machine Learning model
- Predict wine quality
- Evaluate model performance

---

## 📂 Dataset

Dataset: WineQT.csv

The dataset contains different chemical properties of wine, such as acidity, alcohol, pH, sulphates, and quality ratings.
